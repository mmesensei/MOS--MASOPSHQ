# MASOPS Executive Design Canon

## Color System — Founder Decision
**Keep per-executive colors** for in-app differentiation (practical UX), alongside the black-gold brand foundation.

### Brand Palette
| Token | Role | Value |
|---|---|---|
| Background | Page surface | `oklch(0.08 0.008 260)` ≈ `#07080b` |
| Surface | Card/panel | `oklch(0.12 0.010 260)` ≈ `#0d0f14` |
| Border | Dividers | `oklch(0.18 0.008 260)` barely visible |
| Gold primary | All brand accents, names, icons | `oklch(0.72 0.14 75)` ≈ `#c9a227` warm gold |
| Gold glow | Halos, ambient | Gold at ~25% opacity |
| Text primary | Headlines | `#ffffff` |
| Text secondary | Body, roles | `oklch(0.55 0.020 260)` |

### Per-Executive Colors (in-app differentiation only)
- **IRIS** — Violet (Vision/Wisdom) — existing in styles.css
- **APEX** — Blue/Cyan (Structure/Systems) — existing in styles.css
- **KATANA** — Amber/Orange (Execution) — existing in styles.css
- **SENTINEL** — Emerald/Green (Security) — existing in styles.css

These are UI affordances only. In brand contexts (HQ hero, executive panels, loading screens), use gold.

## Typography
- Display/Names: `Space Grotesk`, heavy, uppercase, tracked
- Body: `Inter`, regular, neutral
- Mono: `JetBrains Mono`
- Executive names: heavy uppercase tracked — IRIS, APEX, KATANA, SENTINEL
- Roles: small-caps, spaced — CHIEF EXECUTIVE STRATEGIST
- Taglines: tracked uppercase in gold color — VISION. WISDOM. STRATEGY.
- Brand tagline: maximum tracking — ONE MISSION. ONE SYSTEM. UNLIMITED IMPACT.

## Interface Philosophy
- Premium executive software — command center aesthetic
- Dark surfaces, high contrast, minimal clutter
- Gold accents only where meaningful (not decorative)
- Subtle motion only — state changes, not spectacle
- Glass/depth sparingly and purposefully
- NOT: gaming UI, sci-fi excess, cyberpunk, cartoon, startup clean

## Layout Motifs
- Four-column executive panel with vertical dividers (brand hero)
- Night city skyline through dark glass as environmental backdrop
- Data displays per executive domain:
  - IRIS: Strategic overview charts, global insights
  - APEX: System architecture, geometric lattice holograms
  - KATANA: Operations flow, mission timeline/Gantt
  - SENTINEL: Threat intelligence, world map with nodes

## Brand Identity
- Logo: Gold shield with star/flower motif, MASOPS wordmark
- Tagline: ONE MISSION. ONE SYSTEM. UNLIMITED IMPACT.
- Footer bar: LEARN. │ APPLY. │ GROW. │ LEAD.
